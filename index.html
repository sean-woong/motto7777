About the Art

MOTTO 7777 is a visual system archiving hallucinated identities, corrupted rituals, and glitch-driven personas.
Each figure performs fragments of emotion, caught in an infinite loop of transmission and distortion.
MOTTO 7777은 환각된 정체성과 왜곡된 의식을 기록하는 시각 시스템이다. 각 캐릭터는 감정의 파편을 루프 속에서 반복하며 전송된다.

Process

Created using hand-animated loops (iPad) and distorted visual fragments (DaVinci Resolve), each piece is built from glitch, emotion, and digital decay.
아이패드 루프 애니메이션과 다빈치 디스토션 프레임으로 제작되었으며, 글리치, 감정, 디지털 붕괴로 구성된다.
